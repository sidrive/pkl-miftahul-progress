# Olsera PKL Task Reminder - CLI Tool

Aplikasi CLI kecil berbasis teks untuk memantau status tugas program pendampingan PKL Frontend Olsera.

## Cara Menjalankan

1. Pastikan dependencies sudah terinstall dengan mengetik: `npm install`
2. Jalankan aplikasi menggunakan perintah: `node index.js`

## Dependencies yang Digunakan

1. **chalk** - Digunakan untuk memberikan warna teks pada terminal agar mempermudah pembacaan status tugas.
2. **cli-table3** - Package baru yang berfungsi untuk menyusun format array menjadi bentuk tabel kotak-kotak yang rapi di terminal.
